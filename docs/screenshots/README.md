# Screenshots

Drop your app screenshots in this folder, then they'll appear in the main README.

Suggested images (referenced by the README):

| File name        | What to capture                                  |
| ---------------- | ------------------------------------------------ |
| `home.png`       | The home feed with a few posts                   |
| `post.png`       | A single post showing likes and comments         |
| `register.png`   | The registration page                            |
| `account.png`    | The account / profile page                       |
| `create.png`     | The create-post page                             |
| `banner.png`     | (Optional) A wide hero image for the top         |

Tip: capture at ~1280px wide for crisp images on GitHub.
